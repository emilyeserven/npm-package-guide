<SectionIntro>
Separating config from code. You use <code>.env</code> files for API URLs and feature flags. K8s has ConfigMaps for non-sensitive config and Secrets for sensitive data &mdash; both injectable into your pods.
</SectionIntro>

<SectionSubheading id="toc-frontend">{'\u{1F5A5}\uFE0F'} Your frontend context</SectionSubheading>

<SectionNote>
You use <code>.env</code> files for API URLs, feature flags, and API keys. K8s has ConfigMaps for non-sensitive config and Secrets for sensitive data &mdash; both injectable into your pods.
</SectionNote>

<SectionSubheading id="toc-infra">{'\u2601\uFE0F'} Infrastructure reality</SectionSubheading>

<SectionList>
<ColItem>ConfigMaps store configuration as key-value pairs. Secrets store sensitive data (passwords, tokens, certificates) with base64 encoding. Both can be mounted as environment variables or files inside containers.</ColItem>
</SectionList>
