<SectionSubheading id="toc-start">Where to Start</SectionSubheading>

If you&rsquo;re new to Claude Code&rsquo;s advanced tools, start with the tool that solves your most immediate pain point:

<SectionList>
<ColItem><strong>Spending time re-typing prompts?</strong> Start with <NavLink to="prompt-tools-skills">Custom Slash Commands</NavLink> &mdash; create a single command for your most common task.</ColItem>
<ColItem><strong>Want Claude to access external docs or APIs?</strong> Start with <NavLink to="prompt-tools-mcp">MCP Servers</NavLink> &mdash; the Fetch server alone opens up the web.</ColItem>
<ColItem><strong>Tired of manually formatting AI-generated code?</strong> Start with <NavLink to="prompt-tools-hooks">Hooks</NavLink> &mdash; a PostToolUse hook on Write/Edit auto-formats every change.</ColItem>
<ColItem><strong>Need multi-step, verified workflows?</strong> Start with <NavLink to="prompt-tools-claude-skills">Claude Skills</NavLink> &mdash; encode your process with built-in checks.</ColItem>
<ColItem><strong>Integrating Claude into scripts or CI?</strong> Start with <NavLink to="prompt-tools-optimization">Performance Optimization</NavLink> &mdash; headless mode is the gateway to automation.</ColItem>
</SectionList>
