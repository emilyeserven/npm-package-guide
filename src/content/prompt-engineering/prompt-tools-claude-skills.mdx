<SectionSubheading id="toc-overview">How Skills Work</SectionSubheading>

Skills build on <NavLink to="prompt-tools-skills">custom slash commands</NavLink> by adding structure, verification, and multi-tool orchestration. Where a slash command is a static prompt template, a skill is a complete workflow recipe that tells Claude what to do, how to do it, what tools to use, and how to verify the results.
