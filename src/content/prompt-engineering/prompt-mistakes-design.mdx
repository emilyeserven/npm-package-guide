<SectionIntro>
AI models struggle with visual precision<FnRef n={1} /> ‚Äî they generate hardcoded pixel values instead of responsive units, skip accessibility attributes, drift from design system tokens, and create z-index chaos. The fundamental issue is that AI doesn&rsquo;t see your design specs unless you provide them explicitly. Adding design system constraints to your prompt or CLAUDE.md dramatically improves output quality.
</SectionIntro>

<ColItem><strong>Claude Skills:</strong> Create a <code>/project:a11y-check</code> slash command that audits any component for WCAG compliance and responsive breakpoint coverage.</ColItem>
