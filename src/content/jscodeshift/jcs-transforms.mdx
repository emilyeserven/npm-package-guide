<SectionSubheading id="toc-typescript">TypeScript Support</SectionSubheading>

For TypeScript files, specify the parser when running the codemod:

```bash
# CLI
npx jscodeshift -t transform.ts --parser=tsx src/

# Or set inside the transform:
export const parser = 'tsx'; // 'babel' | 'tsx' | 'babylon' | 'flow' | 'ts'
```

<SectionNote>
The `tsx` parser handles both `.ts` and `.tsx` files. It's the safest default for TypeScript projects.
</SectionNote>
