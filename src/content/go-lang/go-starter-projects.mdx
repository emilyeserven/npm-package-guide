<SectionSubheading id="toc-setup">Setup</SectionSubheading>

<CodeAccordion title="Install & Initialize" ext="bash">
```bash
# Install Go (macOS)
brew install go

# Verify installation
go version

# Create a new project
mkdir my-go-app && cd my-go-app
go mod init github.com/yourname/my-go-app

# Run your code
go run main.go

# Build a binary
go build -o my-go-app
```
</CodeAccordion>
