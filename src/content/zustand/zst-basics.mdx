<SectionIntro>
Zustand (German for "state") is a small, fast, and scalable state management library for React. It uses a simplified flux-inspired approach with hooks ‚Äî no providers, no boilerplate, no context wrappers.
</SectionIntro>

<SectionSubheading id="toc-set">How set Works</SectionSubheading>

The `set` function **shallowly merges** by default ‚Äî you only need to return the properties you want to update.

<ZstCodeBlock exampleId="set-merge" />

<SectionNote>
Pass `true` as the second arg to **replace** the entire state instead of merging: `set(newState, true)`. This is rarely needed but useful for resetting to initial state.
</SectionNote>
