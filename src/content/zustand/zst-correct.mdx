<SectionNote>
**The Golden Rule:** Use Zustand for **client state** (UI toggles, form drafts, user preferences, local filters). Use TanStack Query for **server state** (API data, caching, refetching). Don't put API responses in Zustand unless you have a specific reason.
</SectionNote>
