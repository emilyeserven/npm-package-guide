<SectionIntro>
The Search API finds specific moments within your indexed videos. It supports text queries, image queries, and multiple search modalities that can be combined for precision. Powered by the **Marengo** model.
</SectionIntro>

<SectionSubheading id="toc-examples">Search Examples</SectionSubheading>

<Explainer>
Search supports **36 languages** including Arabic, Chinese, Japanese, Korean, and most European languages. Use semantic search to query in one language and find content in another.
</Explainer>
