<SectionIntro>
Install the JavaScript SDK and initialize the client. You'll need an API key from the TwelveLabs Playground.
</SectionIntro>

<SectionSubheading id="toc-install">Install & Initialize</SectionSubheading>

<CodeAccordion title="Install the SDK" lang="bash" open>
{`npm install twelvelabs-js`}
</CodeAccordion>

<CodeAccordion title="Initialize the client" lang="js" open>
{`const client = new TwelveLabs({
  apiKey: process.env.TWELVELABS_API_KEY
});`}
</CodeAccordion>

<Explainer>
Store your API key in environment variables â€” never hardcode it. In a Vite project, use `import.meta.env.VITE_TWELVELABS_API_KEY`.
</Explainer>

<SectionSubheading id="toc-rest">Direct REST API</SectionSubheading>

If you prefer not to use the SDK, all endpoints are accessible via standard HTTP requests:

<CodeAccordion title="REST API example with fetch" lang="js" open>
{`const response = await fetch("https://api.twelvelabs.io/v1.3/indexes", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": process.env.TWELVELABS_API_KEY,
  },
  body: JSON.stringify({
    indexName: "my-video-library",
    models: [
      {
        modelName: "marengo2.7",
        modelOptions: ["visual", "audio"],
      },
    ],
  }),
});
const index = await response.json();
console.log(index.id);`}
</CodeAccordion>

<Gotcha>
The API uses version **v1.3**. Version 1.2 is deprecated. Also note that the `Generate` API has been renamed to `Analyze` â€” update your code before July 30, 2025.
</Gotcha>
