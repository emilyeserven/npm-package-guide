<SectionSubheading id="toc-what">What is TanStack Start?</SectionSubheading>

If you've been building SPAs with TanStack Router and TanStack Query, Start is the natural next step. It augments the Router rather than replacing it, so you can adopt its server-side features incrementally with minimal disruption.

Start gives you everything you need to go full-stack: server-side rendering, streaming hydration, type-safe server functions, composable middleware, and deployment to any JavaScript runtime â€” all while keeping the client-first DX you already know.
