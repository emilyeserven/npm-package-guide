<Explainer title="Key Architectural Principle">
Unlike frameworks that bundle everything into a monolithic abstraction, Start's layers are **separable**. You can use TanStack Router without Start. You can use Start without any specific deployment target. Each layer adds capability without creating lock-in.
</Explainer>
