<SectionIntro>
Keycloak is an open-source identity and access management (IAM) solution maintained under the CNCF. It handles authentication, authorization, and user management so you don't have to build it from scratch.
</SectionIntro>

<SectionSubheading id="toc-why-care">Why Frontend Engineers Should Care</SectionSubheading>

Keycloak acts as your centralized authentication server. Instead of building login forms, token management, session handling, MFA, social login, and role-based access from scratch, you delegate all of it to Keycloak. Your React app becomes a **relying party** — it redirects users to Keycloak for login and receives secure tokens back.

<SectionSubheading id="toc-strengths">Key Strengths</SectionSubheading>

<SectionList>
  <ColItem>**Single Sign-On (SSO)** — users log in once and access multiple apps via OIDC or SAML</ColItem>
  <ColItem>**Social Login** — built-in Google, GitHub, Facebook, and custom identity provider integration</ColItem>
  <ColItem>**User Federation** — connect to existing LDAP or Active Directory user stores</ColItem>
  <ColItem>**Fine-Grained Authorization** — RBAC (role-based), groups, and policy-based access control</ColItem>
  <ColItem>**Customizable Auth Flows** — visual editor for login steps, MFA, conditional step-up auth</ColItem>
  <ColItem>**Passkeys & WebAuthn** — passwordless authentication (fully supported since v26.4)</ColItem>
  <ColItem>**Admin Console & REST API** — manage everything via UI or programmatically</ColItem>
  <ColItem>**MCP Integration** — can serve as an OAuth2 authorization server for MCP servers (v26.5+)</ColItem>
</SectionList>

<SectionSubheading id="toc-tradeoffs">The Tradeoffs</SectionSubheading>

<Gotcha>
Keycloak is powerful but operationally complex. Self-hosting means you own upgrades, database management, clustering, and security patches. The learning curve is steep — expect to spend time on configuration before it clicks. For smaller projects, managed auth services like Clerk or Auth0 may get you running faster. For enterprise and self-hosted requirements (especially airgapped environments), Keycloak is hard to beat.
</Gotcha>
