<SectionIntro>
Claude Cowork transforms file organization from tedious manual work into a single conversation. Point it at a folder, describe what you want, and step away while Claude sorts, renames, restructures, and cleans up your files.
</SectionIntro>

<SectionSubheading id="toc-what">What is Cowork?</SectionSubheading>

Cowork is a task mode in Claude Desktop that operates on your local files. Unlike Chat (conversational) or Code (programming), Cowork is designed for file management tasks: sorting, renaming, restructuring, deduplicating, and inventorying. It reads file contents to make intelligent decisions about organization.
