<SectionIntro>
**ffprobe** is the diagnostic tool in the FFmpeg suite. Before you transcode anything, you need to understand what you're working with ‚Äî the codec, resolution, bitrate, frame rate, audio channels, and more.
</SectionIntro>

<SectionSubheading id="toc-what">What is ffprobe?</SectionSubheading>

ffprobe reads a video file and reports everything about its structure. It's the first tool you reach for when working with any video file ‚Äî it tells you exactly what's inside before you decide how to process it.

<SectionSubheading id="toc-commands">Common Commands</SectionSubheading>

<CodeAccordion title="ffprobe commands" ext="sh">
<pre><code>{`# Basic info about a file
ffprobe input.mp4

# JSON output (great for scripting)
ffprobe -v quiet -print_format json -show_streams -show_format input.mp4

# Show only video stream info
ffprobe -v quiet -select_streams v:0 -show_entries stream=codec_name,width,height,bit_rate,r_frame_rate input.mp4`}</code></pre>
</CodeAccordion>
