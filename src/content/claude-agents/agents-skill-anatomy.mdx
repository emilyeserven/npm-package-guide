<SectionIntro>
A skill is a directory containing a SKILL.md file with YAML frontmatter and markdown instructions. It can optionally include supporting files like scripts, templates, and reference data. Here's a complete example of a custom skill.
</SectionIntro>

<SectionSubheading id="toc-install">Installing and Distributing Skills</SectionSubheading>

Skills can also be used via the API for programmatic access:

```python
# pip install anthropic
response = client.beta.messages.create(
    model="claude-sonnet-4-5-20250929",
    max_tokens=4096,
    betas=["code-execution-2025-08-25", "skills-2025-10-02"],
    container={
        "skills": [{
            "type": "custom",
            "skill_id": "skill_01AbCd...",
            "version": "latest"
        }]
    },
    messages=[{
        "role": "user",
        "content": "Create a user registration endpoint"
    }],
    tools=[{
        "type": "code_execution_20250825",
        "name": "code_execution"
    }],
)
```
